Facebook Graph API

  • Bietet Applikationen Zugriff auf Nutzerdaten
    • lesend
    • schreibend
  • RESTful basierter Webservice
  • OAuth 2.0 basiertes Zugriffsmodell

Datenstruktur

Struktur der Graph API

Zugriff auf einen Knoten

GET graph.facebook.com
v2.3/{node-id}

Zugriff auf Kanten

GET graph.facebook.com
v2.3/{node-id}/{edge-name}

Struktur der Graph API

Zugriff auf Felder

GET graph.facebook.com
v2.3/{node-id}?fields={field-name, ...}

Suchen nach Knoten

GET graph.facebook.com
v2.3/search?
q={query}&
